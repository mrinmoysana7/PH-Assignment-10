# 🤖 PromptVerse - AI Prompt Sharing Marketplace

PromptVerse is a modern full-stack AI Prompt Sharing Marketplace that empowers creators, professionals, and AI enthusiasts to discover, create, organize, and share high-quality AI prompts. The platform provides a secure role-based environment where users can explore community-generated prompts, publish premium or free prompts, manage collections, and connect with other prompt creators through a clean, responsive, and intuitive interface.

# 💳 Premium Purchase Workflow

PromptVerse uses **Stripe Checkout** for secure premium prompt purchases.

### Purchase Flow

1. Browse Premium Prompt
2. Open Prompt Details
3. Click Purchase
4. Redirect to Stripe Checkout
5. Complete Payment
6. Verify Transaction
7. Unlock Prompt Access
8. Store Transaction Information
9. View Purchased Prompt in Dashboard

Transaction includes:

- Stripe Transaction ID
- Stripe Session ID
- Payment Status
- Purchase Date
